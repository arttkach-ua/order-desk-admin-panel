# Order Desk Admin Panel

React + TypeScript + Vite admin panel with i18n support and a Tailwind-based app shell.

## Quick start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - start Vite dev server
- `npm run build` - type-check and build production assets
- `npm run preview` - preview built app
- `npm run lint` - run ESLint

## Styling stack

- Tailwind CSS is configured via `tailwind.config.js` and `postcss.config.js`
- Global styles and Tailwind directives live in `src/index.css`
- Content pages currently still use MUI components and can be migrated incrementally
